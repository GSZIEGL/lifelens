# LifeLens AI Private V6 – AI képtartalom elemzéssel

Újdonságok:
- Opcionális helyi AI képtartalom-elemzés CLIP modellel.
- Nem csak fájlnév/mappa alapján kategorizál, ha bekapcsolod az AI-t.
- Videóknál a preview képkockát is tudja elemezni.
- Normálisabb albumválogatás: kategória + minőség + napi változatosság.
- Látványos HTML fotókönyv/idővonal export.
- Helyi/online privát jelzés megtartva.

AI bekapcsolásához:
```bash
pip install torch transformers
```

Alap módban ezek nélkül is fut.
